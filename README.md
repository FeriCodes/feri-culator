# FeriCulator
FeriCulator: A simple Python calculator featuring persistent history and error handling.

Hi! This is my first Python project. It's a simple calculator that saves your calculation history to a local JSON file with precise timestamps.

## Features
- **Basic Math**: Supports +, -, *, /, %, //, and ** (power).
- **Square Root**: Calculate `sqrt` of numbers.
- **History**: Automatically saves every calculation to `CalHistory.json`.
- **History Management**: View your past calculations or clear the history file.
- **Error Handling**: Prevents crashes from invalid inputs or division by zero.
- **Modular Architecture**: Built with clean Python classes and packages.

## How to Run
1. Make sure you have Python installed.
2. Clone the repository and navigate to the project directory.
